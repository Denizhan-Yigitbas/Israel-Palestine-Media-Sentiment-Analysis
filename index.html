Exploring Sentiment Analysis of News Headlines

Special Focus: Moving the US Embassy from Tel Aviv to Jerusalem

By Denizhan Yigitbas

Question

In recent years, data analytics has been growing as a method to give us a better understanding of the world in all different sorts of industries. In the world of politics, sentiment analysis has been developing as a branch of data science that allows researchers to observe whether specific statements can be classified as a positive, negative, or neutral statement. One of the most popular areas sentiment analysis has been implemented is Twitter. Twitter is a platform that allows users to freely write about any topic they would like. Sentiment analysis comes into play in various scenarios. For example, a company might want to understand how customers are feeling about a certain product. If Apple was interested in to see how their customers feel about the new iPhone, they could extract tweets contain the hashtag "#iphone", perform a sentiment analysis on each tweet, and determine if the majority is positive, negative, or neutral. In politics, researchers can run a sentiment analysis on tweets posted by presidential candidates to understand the effect the sentiment has on polls or other outcomes. In this study, the goal was to experiment if the same approach of sentiment analysis could be used in reference to news headlines. Do different news sources tend to present specific topics in a consistent sentiment to fit their audience? This study specifically focuses on how news sources differ in their composition of news headlines when publicizing articles specifically about the United States' embassy move from Tel Aviv to Jerusalem. Is it possible to attribute a positive, negative, or neutral sentiment to the headlines designated for this event? In order to explore these questions, 4 different new outlets will be analyzed: The New York Times, Fox News,Mondoweiss, and HonestReporting.

Hypothesis

In order to proceed with the analysis, we must define what is meant by positive, negative, and neutral sentiment in the context of moving the US Embassy from Tel Aviv to Jerusalem. Part of the hypothesis assumes that a positive sentiment in a news headline about this event implies support of the move. This can be for reasons that benefit Israel or the US in any way. On the other hand, a negative sentiment within a headline implies being against the move of the embassy. For example, this can be due to the negative impacts could potentially have for Palestinians in the area. Proceeding with these assumptions, it is hypothesized that right-leaning platforms such as Fox News and HonestReporting will cover the move of the embassy with more positive sentiment, while left-leaning platforms such as The New York Times and Mondoweiss will cover the move with more negative sentiment.

Methodology

Sentiment analysis can be performed in various ways, the most simple of which is called Lexicon-Based Matching. There are 3 mains parts to Lexicon-Based Matching: Data Extraction, Data Cleaning, and Data Processing. In order to complete all parts, Python was used as the main backend software integrated with various APIs and libraries to run the analyses. Once the necessary data processing was completed, excel and R were utilized to visualize the results.

Data Extraction

The first step of any data science research is collecting the data that will be analyzed. In the context of this project, we must aim to collect news headlines about the US Embassy move from Tel Aviv to Jerusalem from The New York Times, Fox News, Mondoweiss, and HonestReporting. Large platforms such as The New York Times and Fox News provide public APIs that allow developers to extract desired data from these sources with simple software. For this reason, a python wrapper on these APIs allowed for the extraction of around 50 headlines from each source that were the most relevant for the query "US Embassy move from Tel Aviv to Jerusalem". Smaller platforms such as Mondoweiss and HonestReporting do not provide such APIs for this reason, the same query was used for these sources, however the most relevant headlines were manually extracted.

Data Cleaning

Lexicon-Based Matching analysis by removing or modifying any irrelevant, duplicated, or incorrect data. Oftentimes when performing sentiment analysis on product reviews are tweets, this step could be very involved for many reasons. For example, if tweets were being used, links, hashtags, mentions, symbols, etc would need to be removed from the text because they do not provide any additional information. However, the data set used in the project are news headlines, implying that they are professionally written and contain only letters, numbers, and punctuation. For this reason, the only significant data cleaning that needs to be done is the removal of what are called stop words. Stop words include determiners, conjunctions and prepositions such as “a”, “for”, “but”, “in”, etc. Following this, in order to avoid any additional errors in the processing section, we will convert every character of every headline to be coded in UTF-8, allowing for data consistency when performing Natural Language Processing for Sentiment Analysis.

Data Processing

Now that the necessary data has been extracted and cleaned, it is ready to undergo the sentiment analysis process. The basics of sentiment analysis can be outlined as a five step process: Tokenize the input into component sentences based on punctuation This can be completed with a simple string manipulation algorithm that detects spaces and punctuation and divides them accordingly Tag each token with a part-of-speech using a natural language processing algorithm This can be completed utilizing the NLP algorithms provided by the used library Assign quantitative values to each token by matching them to a word lexicon Each token will be matched with the corresponding word in a word lexicon. Once each token is matched to a quantitative value, a statistical average will be taken for the overall input Output a normalized polarity for the input between -1 and 1 In order to compare results, the averages will be normalized to values between -1 and 1 Map the polarity of each input to a sentiment Any polarity value greater than 0 equates to positive sentiment. Similarly, any polarity value less than 0 equates to a negative sentiment. Finally, if the polarity is equal to zero, this implies a neutral sentiment. In order to ensure accuracy of the results, the following 3 natural language processing libraries for sentiment analysis were used: TextBlob, Vader, and SentimentR. Using 3 different libraries allows for the use of 3 different lexicons. In order to make an accurate conclusion, we would expect the results to be similar amongst the 3 libraries. While TextBlob and Vader are NLP libraries written in python, SentimentR is a library written solely for R software. Various data transformation methods were used in order to combine the resulting data processing for all 3 libraries. Once combined, the data is prepared to be analyzed for results.

In each graph, the x-axis will contain the 3 various natural language processing libraries that were used for sentiment analysis: TextBlob, Vader, and SenimentR.
